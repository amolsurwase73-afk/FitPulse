### Overview
FitPulse is a lightweight fitness app that tracks daily steps, water intake, mood, and workouts. It provides AI-powered workout suggestions, hydration reminders, and weekly progress charts.

### Features
- Track steps, water intake and mood
- AI-generated quick workouts
- Offline logging with auto sync
- Weekly progress charts and insights
